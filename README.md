# Catálogo AGBooks

Um catálogo online de livros interativo, permitindo que usuários vejam livros, adicionem ao carrinho e escolham mostrar ou esconder preços. Ideal para projetos de e-commerce ou demonstrações de front-end.

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Layout responsivo
- Contador de carrinho interativo
- Botão para mostrar/esconder preços

---

## 📖 Funcionalidades

1. **Visualização de livros**  
   Cada livro é exibido com imagem, título, autor e preço.

2. **Adicionar ao carrinho**  
   Usuários podem adicionar livros ao carrinho, com contador visual na tela.

3. **Mostrar/Esconder preços**  
   Um botão permite alternar a visibilidade dos preços de todos os livros.

4. **Responsivo**  
   Layout adaptável para diferentes tamanhos de tela, incluindo dispositivos móveis.
